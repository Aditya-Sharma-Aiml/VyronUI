                        VYRONUI — PROJECT REQUIREMENTS
               Full-Stack AI-Powered React UI Component Platform
================================================================================

PROJECT OVERVIEW
----------------
VyronUI is a full-stack SaaS web platform that lets users:
  1. Browse and use a prebuilt React UI component library (vyron-ui-library on npm)
  2. Generate custom React UI components using AI (DeepSeek via OpenRouter)
  3. Preview, customize, and save those components
  4. Purchase AI credits via Razorpay
  5. (Admin) Publish components to the npm library with one click

The project is divided into 3 sub-packages:
  - vyron-ui-client   → React frontend (Vite + TailwindCSS v4)
  - vyron-ui-server   → Node.js/Express backend API
  - vyron-ui-lib      → Publishable npm component library (tsup build)

3. SERVER — vyron-ui-server
================================================================================

  Framework      : Express 5
  Language       : JavaScript ESM ("type": "module")
  Database       : MongoDB via Mongoose 9
  Auth           : JWT (jsonwebtoken) + httpOnly cookies
  AI             : OpenRouter API → DeepSeek V3 (deepseek/deepseek-chat)
  Payments       : Razorpay

  --- Dependencies ---

  express                       ^5.2.1     — HTTP server framework
  mongoose                      ^9.2.4     — MongoDB ODM
  dotenv                        ^17.3.1    — Environment variable loader
  cors                          ^2.8.6     — CORS (origin: http://localhost:5173)
  cookie-parser                 ^1.4.7     — Parse httpOnly JWT cookies
  jsonwebtoken                  ^9.0.3     — JWT sign + verify
  bcryptjs                      ^3.0.3     — Password hashing (future use)
  axios                         ^1.13.6    — HTTP client (OpenRouter API calls)
  crypto                        ^1.0.1     — HMAC signature for Razorpay verification
  razorpay                      ^2.9.6     — Razorpay Node SDK
  nodemon                       ^3.1.14    — Dev auto-restart

  --- Environment Variables (.env) ---

  PORT                          = 8000
  MONGODB_URL                   = <MongoDB connection string>
  JWT_SECRET                    = <JWT signing secret>
  OPENROUTER_API_KEY            = <OpenRouter API key>
  RAZORPAY_KEY_ID               = <Razorpay Key ID>
  RAZORPAY_KEY_SECRET           = <Razorpay Key Secret>

  --- API Routes ---

  POST  /api/auth/googlesignup       — Firebase token → create/login user, set JWT cookie
  GET   /api/auth/logout             — Clear JWT cookie

  GET   /api/user/currentuser        — Get logged-in user data (isAuth)
  GET   /api/user/all-users          — Get all users (isAuth)

  POST  /api/component/generate      — AI component generation (isAuth, costs 50 credits)
  POST  /api/component/save          — Save component to DB (isAuth)
  POST  /api/component/publish       — Publish component to npm (isAuth, admin only)
  GET   /api/component/all-components — Get all components (isAuth)

  POST  /api/payment/create-order    — Create Razorpay order (isAuth)
  POST  /api/payment/verify          — Verify payment + add credits (isAuth)

  --- Data Models ---

  User
    name          String (required)
    email         String (required, unique)
    role          String enum ["user", "admin"] (default: "user")
    aiCredits     Number (default: 150)

  Component
    name          String
    code          String (full JSX code)
    props         [String] (customizable prop names)
    owner         ObjectId → User
    visibility    String enum ["private", "public"] (default: "private")
    npmPackage    String (set to "vyron-ui-lib" on publish)

  Payment
    userId        ObjectId → User (required)
    amount        Number (INR)
    aiCredits     Number (credits being purchased)
    razorpayOrderId   String
    razorpayPaymentId String
    status        String enum ["created", "paid", "failed"] (default: "created")

  --- Middleware ---

  isAuth.js      — Verifies JWT from cookie → sets req.userId

  --- AI Integration ---

  Provider       : OpenRouter (https://openrouter.ai/api/v1/chat/completions)
  Model          : deepseek/deepseek-chat (DeepSeek V3)
  Temperature    : 0.7
  Max Tokens     : 2000
  Response Format: { type: "json_object" } (enforced JSON output)
  Credit Cost    : 50 credits per generation (admin generates free)

  --- Publish Pipeline (Admin only) ---

  When admin publishes a component:
    1. Write JSX file to vyron-ui-lib/src/components/<Name>/<Name>.jsx
    2. Append export to vyron-ui-lib/src/index.js
    3. Delete old dist/ folder
    4. Run: npm run build (tsup)
    5. Run: npm version patch
    6. Run: npm publish --access public
    7. Update component.visibility = "public" in DB

  --- Component Rules (AI-generated components follow these) ---

  - Named export only: export const ComponentName = ({ ...props }) => {}
  - Inline styles ONLY (no TailwindCSS, no CSS classes)
  - All props have default values
  - No external dependencies (only React + hooks)
  - No position: fixed (breaks live preview sandbox)
  - Dark background by default (#0f172a, #020617, #0d1117)
  - Rich accent colors: #6366f1, #7c3aed, #059669, #e11d48, #0ea5e9

6. DEV SETUP COMMANDS
================================================================================

  # Install all dependencies
  cd vyron-ui-client  && npm install && npm install framer-motion
  cd vyron-ui-server  && npm install
  cd vyron-ui-lib     && npm install

  # Start development
  cd vyron-ui-client  && npm run dev       # → http://localhost:5173
  cd vyron-ui-server  && npm run dev       # → http://localhost:8000

  # Build & publish library
  cd vyron-ui-lib && npm run build && npm publish

  # Install published library in client
  cd vyron-ui-client && npm i vyron-ui-library

7. KNOWN ISSUES / NOTES
================================================================================

  1. CRITICAL — framer-motion missing:
     package.json has "motion" ^12.35.2 but all source files import from
     "framer-motion". Run: npm install framer-motion in vyron-ui-client.

  2. Firebase API Key empty:
     .env has VITE_FIREBASE_APIKEY= (empty). Fill in Firebase console
     project key or Google OAuth login will silently fail.

  3. All .env files are empty/placeholder:
     server/.env: MONGODB_URL, JWT_SECRET, OPENROUTER_API_KEY,
     RAZORPAY_KEY_ID, RAZORPAY_KEY_SECRET all need real values.
     client/.env: VITE_FIREBASE_APIKEY, VITE_RAZORPAY_KEY_ID need real values.

  4. npm library name mismatch:
     Package is published as "vyron-ui-library" but some UI text says
     "npm install vyron-ui-lib". Standardize to "vyron-ui-library".

  5. Server CORS is hardcoded:
     origin: "http://localhost:5173" — update for production deployment.

  6. Publish pipeline runs synchronously on the HTTP thread (execSync):
     Heavy operation — may timeout in production. Consider async job queue.
